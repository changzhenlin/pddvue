# 多多移动商城 Demo

一个使用 Vue 3 实现的拼多多风格移动端纯前端电商 Demo。项目数据均为本地 mock，覆盖从首页浏览、活动页拼单、商品详情、购物车、结算到订单成功的完整购物流程。

## 功能

- 首页：红色促销头部、搜索入口、频道导航、限时秒杀、商品双列流。
- 活动页：优惠券、倒计时氛围、活动商品和立即拼单入口。
- 商品详情：规格选择、数量调整、加入购物车、立即购买。
- 购物车：勾选商品、全选、数量增减、删除、合计和去结算。
- 结算页：默认地址、商品明细、优惠金额、实付款和提交订单。
- 成功页：展示 mock 订单号和支付金额。

## 技术栈

- Vue 3
- Vite
- Vue Router
- Pinia
- 本地 mock 数据

## 本地运行

```bash
npm install
npm run dev
```

默认访问地址：

```text
http://127.0.0.1:5173/
```

## 构建

```bash
npm run build
```

## 项目结构

```text
src/
  components/        通用组件：底部导航、商品卡片、数量步进器
  data/              mock 商品、频道、优惠券和地址数据
  router/            页面路由
  stores/            Pinia 购物流程状态
  styles/            全局样式
  views/             首页、活动页、详情页、购物车、结算页、成功页
```

## 购物流程

```text
首页/活动页 -> 商品详情/立即拼单 -> 购物车 -> 结算页 -> 订单成功页
```

活动页商品也支持直接进入结算页完成下单。
